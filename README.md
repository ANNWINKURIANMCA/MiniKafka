# MiniKafka

A lightweight Kafka-inspired distributed message broker built in Java.

## Features

- Producer
- Consumer
- Topic-based messaging
- Persistent log storage
- Message offsets
- Partitions
- Consumer groups
- Docker support
- GitHub Actions (Coming Soon)

## Tech Stack

- Java 21
- TCP Sockets
- Docker
- GitHub Actions
- AWS (Planned)

## Project Structure

```
src/
broker/
producer/
consumer/
storage/
common/
```

## Run

```bash
docker compose up --build
```
